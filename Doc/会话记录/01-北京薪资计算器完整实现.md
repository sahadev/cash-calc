# 北京薪资计算器完整实现

> **提问**：在当前目录下新建一个 React+TS+Tailwind 工程，输入月 base 可计算北京市五险一金和税，算出每月到手现金、养老金和公积金，支持年终奖计算，可看详细信息并记录存档。

---

## 回答摘要

### 技术栈
- Vite + React 18 + TypeScript + Tailwind CSS v4
- 纯前端计算，localStorage 持久化

### 实现的核心功能

| 模块 | 说明 |
|------|------|
| 输入表单 | 月base、薪月数、公积金比例(5~12%)、专项附加扣除、年终奖计税方式 |
| 计算引擎 | 五险一金双向计算、累计预扣法逐月个税、年终奖单独/并入对比 |
| 逐月明细 | 12个月可展开查看4步详细计算过程 |
| 年度汇总 | 到手现金、养老金、公积金、综合价值 |
| 历史存档 | localStorage 存档/加载/删除 |

### 北京 2025 年度政策参数
- 社保基数: 7,162~35,811 元
- 公积金基数: 2,540~35,811 元
- 个人缴纳: 养老8% + 医疗2% + 失业0.5% + 公积金5~12%
- 个税: 累计预扣法七级税率表，起征点5000元/月

### 验算结果（月base 10000, 15薪）
- 五险一金个人: 2,250/月 → 27,000/年
- 工资个税: 990/年
- 年终奖个税: 900（单独计税推荐）
- 到手现金: 121,110 元/年
- 综合价值: 178,710 元/年

## 关键结论

- 计算引擎通过两个不同薪资档位验证，累进税率跳档逻辑正确
- 年终奖自动对比两种计税方式并推荐最优方案
- TypeScript 类型检查和生产构建均通过
- 深色金融仪表盘风格 UI
